
# io_el3d

## Introduction

C'est un module pour permetre l'importation (et bientot l'exportation) des fichiers .e3d de landes eternelles sur blender 3+.

## Installation

Cliquer sur le bouton code (vert) en haut as droit de cette page. Puis en bas dans la bulle qui apparait sur Download ZIP.

Ensuite rendez vous sur Blender:
- Editer
- Préférence...
- Sur la gauche Add-ons
- puis vers le haut as droite installer
- dans Telechargement selectionnez io_el3d-main.zip
- puis install Add-on

Dans la fenêtre Préférence de Blender - Add-ons:
- En haut selectionner **En test**
- **Enabled Add-ons Only** doit être decoché
- Le selecteur de categorie doit etre sur **Tout** ou **Import-export**

Cochez **Import-Export: io_el3D** pour activer le module.

## Utilisation

Vous pouvez importer les modeles en passant par le menu Fichier->Import->EL3D Import Operator.
